
# 🏋️ Gym Management System (MySQL)

## 📌 Overview

A complete Gym Management Database built using MySQL to manage members, trainers, subscription plans, payments, and attendance efficiently.

---

## 🚀 Features

* 👤 Member Management
* 🏋️ Trainer Management
* 💳 Subscription Plans
* 💰 Payment Tracking
* 📅 Attendance Tracking (Auto via Trigger)
* 📊 Active Members View

---

## ▶️ How to Run

1. Open MySQL Workbench
2. Create database:

```sql
CREATE DATABASE gym_db;
USE gym_db;
```

3. Open `gym_project.sql`
4. Run all queries
