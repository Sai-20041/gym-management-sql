# 🏋️ Gym Management System (MySQL)

## 📌 Project Overview

This project is a Gym Management System built using MySQL. It manages members, trainers, subscriptions, payments, and attendance.

## 🚀 Features

* Member & Trainer Management
* Subscription Tracking
* Payment System
* Attendance Tracking (Trigger-based)

## 🧠 SQL Concepts Used

* DDL (CREATE DATABASE, TABLE)
* DML (INSERT, SELECT)
* Joins (INNER JOIN)
* Subqueries
* Views
* Stored Procedures
* Triggers
* Transactions
* Indexing

## 🗂️ Tables

* members
* trainers
* plans
* subscriptions
* payments
* attendance

## ▶️ How to Run

1. Open MySQL Workbench
2. Create database:

   ```sql
   CREATE DATABASE gym_db;
   USE gym_db;
   ```
3. Run the SQL file:

   * Open `gym_project.sql`
   * Execute all queries

## 📊 Sample Queries

* Active Members
* Monthly Revenue
* Trainer-wise Member Count
